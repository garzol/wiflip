Dimensions

The RS3 CLONE V1B (RS3CLN1B) MPU card has exactly the same dimensions as the original MPU card in the Recel system III. It has the same notches on the edges to allow the card to be fixed in the same way as the original (with the original spacers).

The board may be installed in the same way has the original, or directly screwed on the backplane, since it does not need any thermal dissipator.

LxH=260mmx178mm
